# CPA問題テキスト整形 指示書

## 目的

OCRで読み取ったCPA短答式○×問題のテキストを、アプリにインポートできる形式に整形してください。

## 出力フォーマット（厳守）

以下の構造で出力してください。余計な説明・コメントは不要です。整形済みテキストのみを出力してください。

```
[セクションヘッダー]
N.セクション名

[問題]
ロロロ 問題N 重要度X [年度情報]
問題文（1行にまとめる）

[解答]
ロロロ 問題N 正しい／誤り
解説文（1行にまとめる）

[補足]（あれば）
追加でチェック！
補足説明文（1行にまとめる）
```

### 具体例

```
1.真実性の原則
ロロロ 問題1 重要度B
企業会計は，企業の財政状態及び経営成績に関して，真実な報告を提供するものでなければならないとする原則は企業会計の最高規範として位置付けられるが，この原則において，唯一絶対の真実性が求められているわけではない。
ロロロ 問題2 重要度A H22過去問
問題文...
2.正規の簿記の原則
ロロロ 問題3 重要度B
問題文...

ロロロ 問題1 正しい
解説文...
追加でチェック！
補足説明文...
ロロロ 問題2 誤り
解説文...
ロロロ 問題3 誤り
解説文...
追加でチェック！
補足説明文...
```

## 整形ルール

### 1. 構造

- **前半に問題パート**、**後半に解答パート**をまとめる
- 原文ではページごとに問題と解答が交互に出てくるが、**すべての問題を先に、すべての解答を後に並べ替える**
- 問題番号は原文のまま維持（章ごとに1から始まる）
- 問題パートと解答パートの間には**空行を1行**入れる

### 2. セクションヘッダー

- `N.セクション名` の形式（例: `1.真実性の原則`）
- セクションヘッダーは**問題パート側にのみ**記載する（解答パート側には不要）
- 半角数字 + 半角ピリオド + セクション名
- ページ区切りで重複して出てくるヘッダー（例: 毎ページ冒頭の「2．一般原則」）は**削除する**

### 3. 問題行

形式: `ロロロ 問題N 重要度X [年度情報]`

- `ロロロ` は全角カタカナのロ3文字（半角スペース区切り）
- 重要度は `A`（難）/ `B`（標準）/ `C`（易）
- 年度情報がある場合はそのまま残す
  - 平成 → `H` + 数字（例: H22過去問、H30過去問類題）
  - 令和 → `R` + 数字（例: R4過去問）
  - `過去間` は OCR 誤読なので `過去問` に修正

### 4. 解答行

形式: `ロロロ 問題N 正しい` または `ロロロ 問題N 誤り`

- 「正しい」「誤り」の2択のみ
- 解答行の後ろに `「企業会計原則」第一 一` のような出典情報がある場合は**削除する**（解説文に含めれば十分）

### 5. 補足（追加でチェック）

- `追加でチェック！` の行で始まる（`！`は全角）
- `追加でチェック!`（半角!）や `追加でチェック1`（数字の1）はOCR誤読 → `追加でチェック！` に修正
- 年度情報（例: `H28過去問`）が付いている場合はその行ごと `追加でチェック！` に統一
- 補足説明文は次の行に記載

### 6. テキスト整形

- **改行の結合**: OCRで途中改行されている文は1行にまとめる
- **句読点の統一**: `，`（全角カンマ）と `。`（全角句点）を使用
- **不要な記号の除去**: 行末の `'` `"` `.` や、OCRノイズの `㈱` 等を除去
- **ページ番号の削除**: `①-3` `②-1` のような丸数字ページ番号の行は完全に削除
- **章ヘッダーの削除**: `第N章 ...` のような行は削除（セクションヘッダーとは別）
- **単独の「問題」行の削除**: ページ先頭に出る単独の「問題」という行は削除

### 7. OCR誤読の修正

以下は頻出するOCR誤読パターンです。**必ず修正**してください。

#### 記号・マーカー系
| OCR誤読 | 正しい表記 |
|---------|-----------|
| nnn, nnD, nrln, DDD, DD口, Dn口, i~1nn, l~1nn | ロロロ |
| 膜り | 誤り |
| 娯り | 誤り |
| 過去間 | 過去問 |

#### 漢字の誤読（頻出）
| OCR誤読 | 正しい表記 | 出現箇所の例 |
|---------|-----------|-------------|
| 恋意性 | 恣意性 | 経営者の恣意性 |
| 盗本 | 資本 | 払込資本 |
| 街本 | 資本 | その他資本剰余金 |
| 会計帳薄 | 会計帳簿 | 正確な会計帳簿 |
| 記戦 | 記載 | 貸借対照表に記載 |
| 猿得 | 獲得 | 企業が獲得した |
| 饗用 | 費用 | 収益及び費用 |
| 資産・負侵 | 資産・負債 | |
| 掲戦 | 掲載 | |
| 開示制皮 | 開示制度 | |
| 処理方浩 | 処理方法 | |

#### その他
- `㈱` → 文脈に応じて削除または「株式会社」に修正
- 半角の `,` `.` が紛れている場合 → 全角 `，` `。` に統一
- `会計･処理` のような中黒の誤挿入 → `会計処理` に修正

### 8. やってはいけないこと

- 問題文や解説文の**内容を変更・要約・省略しない**（OCR誤読の修正のみ行う）
- 問題番号を勝手に振り直さない
- 存在しない問題や解答を追加しない
- 重要度を変更しない
- 自分の判断で「正しい」「誤り」を変更しない

## 入力テキストの構造について

入力は以下のような構造になっています:

1. 各ページの先頭に丸数字ページ番号（`①-1`、`②-3` 等）
2. 左ページ（奇数）に問題、右ページ（偶数）に解答
3. ページごとに章名ヘッダー（`2．一般原則` 等）が繰り返し出現
4. 問題と解答が交互に配置されている

これを**問題→解答の順にまとめ直して**ください。

## チェックリスト

出力前に以下を確認してください:

- [ ] すべての問題に対応する解答があるか
- [ ] 問題番号が連番になっているか
- [ ] セクションヘッダーが正しい位置にあるか
- [ ] OCR誤読（ロロロ、誤り、過去問 等）が修正されているか
- [ ] ページ番号行が削除されているか
- [ ] 重複する章ヘッダーが削除されているか
- [ ] 改行が適切に結合されているか
